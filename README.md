# GB Minesweeper

初代ゲームボーイ向けのシンプルなマインスイーパーです。

RGBDSによるゲームボーイ開発の習得を目的とし、入力、描画、乱数、サウンドなどの処理は、将来のゲームボーイRPG開発にも流用できる構成を目指します。

## 開発状況

現在は開発初期段階です。プロジェクト構成、ビルド方法、ゲーム仕様には今後変更が入る可能性があります。

## ゲーム仕様（予定）

- 対応機種: 初代ゲームボーイ
- 盤面: 9×9マス
- 地雷: 10個
- クリア条件: 地雷以外のすべてのマスを開く
- ゲームオーバー条件: 地雷のあるマスを開く

## 操作方法（予定）

- 十字キー: カーソル移動
- Aボタン: マスを開く
- Bボタン: フラグを設置・解除
- STARTボタン: ゲーム開始／リスタート

## 開発環境

- WSL2 Ubuntu
- Visual Studio Code + Remote WSL
- RGBDS
- SameBoy / BGB
- hUGETracker（BGM・効果音）

ビルド手順と実行方法は、最小ROMとプロジェクト構成の完成後に追記します。
